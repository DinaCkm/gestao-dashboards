# Guia de Cálculo de Performance - Sistema de Mentorias CKM Talents

## Visão Geral da Performance

A "Visão Geral" exibida no painel individual é calculada por meio da **média simples de cinco indicadores principais**:

**Fórmula:**
```
Visão Geral = (Indicador1 + Indicador2 + Indicador3 + Indicador4 + Indicador5) / 5
```

## Os 5 Indicadores

### 1. Participação nas Mentorias
Percentual de encontros de mentoria em que o participante esteve presente.

**Fórmula:**
```
Participação nas Mentorias = (Mentorias com presença / Total de mentorias) × 100
```

### 2. Atividades Práticas
Quantidade de atividades práticas entregues em relação ao total de atividades previstas no ciclo.

**Fórmula:**
```
Atividades Práticas = (Atividades entregues / Total de atividades previstas) × 100
```

### 3. Evolução / Engajamento
Indicador qualitativo e quantitativo que mede o **nível de envolvimento ativo** do participante ao longo dos meses, considerando:
- Presença nas mentorias
- Entrega de atividades
- Participação com constância e proatividade

O engajamento é atribuído mensalmente como **nota de 0 a 10**, depois convertida para percentual.

**Fórmula:**
```
Engajamento = (Nota de Engajamento / 10) × 100
```

**Importante:** Não é só presença! O painel considera interação, constância e entrega.

### 4. Performance das Competências
Refere-se ao **quanto o participante estudou** das trilhas de aprendizagem e à **nota obtida na autoavaliação** feita após cada trilha.

**Regras:**
- São consideradas apenas as competências **cujas aulas já foram liberadas**
- Competências com 0% são **ignoradas** no cálculo se não estiverem disponíveis
- A média final considera **apenas as trilhas disponíveis + notas das autoavaliações**

### 5. Participação nos Eventos
Reflete a presença em eventos como **webinários** e **encontros coletivos** registrados no programa.

**Fórmula:**
```
Participação em Eventos = (Eventos com presença / Total de eventos realizados) × 100
```

## Observações Finais
- Todos os cinco indicadores têm **o mesmo peso** na média final da performance
- O objetivo é oferecer uma visão **equilibrada entre presença, entrega, aprendizado e envolvimento**

## Estágios de Performance (Notas do Aluno)

| Estágio | Nota | Descrição |
|---------|------|-----------|
| **Excelência** | 9-10 | Alto nível de engajamento com a plataforma, webinars, tarefas da mentoria, aplicação dos aprendizados e evidente evolução das competências indicadas para o desenvolvimento no Diagnóstico |
| **Avançado** | 7-8 | Bom nível de engajamento com a plataforma, webinars, tarefas da mentoria, aplicação dos aprendizados e evidente evolução das competências indicadas para o desenvolvimento no Diagnóstico Inicial |
| **Intermediário** | 5-6 | Comprometimento adequado, com pouco equilíbrio na realização entre as ferramentas disponibilizadas como a plataforma, webinars, tarefas da mentoria, aplicação dos aprendizados e com uma evolução regular das competências indicadas para o desenvolvimento no Diagnóstico Inicial |
| **Básico** | 3-4 | Participação irregular, progresso limitado |
| **Inicial** | 0-2 | Baixo engajamento, pouca evolução |
